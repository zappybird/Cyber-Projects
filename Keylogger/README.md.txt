# Keylogger

This is a simple keylogger written in Python using the `pynput` library. It logs keystrokes to a specified text file.

## Files

- `config.py`: Contains configuration settings.
- `logger.py`: Handles logging of keystrokes.
- `main.py`: The main entry point of the application.
- `test_logger.py`: Contains unit tests for the logging functionality.

## Usage

To run the keylogger, execute the `main.py` file:

```bash
python main.py
